Gamerz Corner

Project Gamerz Corner is a full-stack social networking application that also utilizes an Application Programming Interface. The purpose of Gamerz Corner is to bring the gaming community together to socialize as well as have a platform to organize and review games from all eras. Visitors to the page can view games, reviews, and read message boards pertaining to a broad range of topics. Visitors who register and become members can create, edit and delete message boards as well as customize their profile page with a list of games they have played and reviewed and make connections with other registered members.

Restblog Project

Restblog is a full-stack single-page application using MariaDB and MySQL for database storage. This application allows users to register, login, and create posts as well as edit and delete a post.

Movie Project

The Movies project was a single-page application developed using paired programming. The movies project utilizes a mock database as well as the Open Movie Database API. A visitor to the page can add, edit, rate, and delete movies. This application was constructed using HTML, CSS, and Bootstrap frameworks for the front-end in addition to Javascript and JQuery for backend calls to the API.

Weather Map Project

The weather map project was independently built and uses the OpenWeatherMap and Mapbox APIs. This application allows users to acquire a five-day weather forecast based on the desired location using a dynamic weather map with markers and weather cards that correspond with geolocation inputs such as longitude/latitude, an address, or landmark names. This project was developed using HTML, CSS, and Bootstrap frameworks to generate page elements on the front-end while Javascript and JQuery were used to make ajax requests to the APIs.

Coffee Project

Collaborated with a partner to create a coffee directory which permits customers ease of use in searching by name and roast types. This application allows for addition of new products as well as predictive text using active filtration. The project was built using HTML, CSS, and javascript. Jquery was utilized to allow for domain object model manipulation.
